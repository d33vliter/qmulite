
# QmuLite

Script para ejecutar maquinas virtuales con Qemu-KVM.


Pasos:
volver ejecutable el archivo "requisitos" y "qmulite":

`chmod +x requisitos`

`chmod+x qmulite`

Ejecutar `./requisitos` para instalar las dependencias(sudo)

Y pa podras ejecutar QMULITE con `./qmulite`
